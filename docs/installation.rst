.. highlight:: shell

============
Installation
============

At the command line::

    $ python setup.py install
    

To create a virtual environment for IMM using Python 3.3 or better do::

	$ pyvenv IMM_VENV
	$ IMM_VENV\Scripts\activate
	(IMM_VENV) $ pip install pillow
	(IMM_VENV) $ pip install imm
